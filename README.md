# Belgarbeit Webentwicklung

Ein Onlineshop für Kleidung, entstanden im Kurs Webentwicklung an der HTW Berlin.

## Getting Started

Führe die folgenden Befehle aus :

<code>
$ yarn install
</code>

<code>
$ bundle install
</code>

<code>
$ rails db:migrate
</code>

Optional um die Datenbank mit Beispieldaten zu füllen

<code>
$ rails db:seed
</code>
